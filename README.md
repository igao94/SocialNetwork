# SocialNetwork

Welcome to SocialNetwork, a mini app where users can connect with others, create posts, add likes and comments, and more.

Getting Started

Create User: Begin by creating a user account using your email and password. Log In: Log in with your email and password to access the app. Authorization: Upon successful login, you will receive a token. Copy this token and use it for authorization. Authorize: In Postman or any API client, use the received token for authorization.

Admin Access Default Admin: Username: Igor Email: igor@gmail.com Password: 123456 To access admin functionalities and profile, log in with the provided admin credentials. Admin actions include:

Deleting Users: Admin can delete user accounts if deemed necessary.
Deleting Posts: Admin can remove posts(likes and comments also) from the platform if necessary.
Additional Notes Reporting: Users can report other users or posts, but only the admin can view those reports. Data Export: Users can request all their data in the form of a CSV file. Postman Collection: A Postman collection is available for every endpoint for easy testing and usage.
